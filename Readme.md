# Introduction

1:这个项目是使用阿里的egg 框架做的web项目;  
2:使用mysql 做数据库 ;  
3:使用egg-view-ejs 做模板;  
4:运行环境是node v8.11.3;  
5:这个项目本来是公司的官网 后面因某些原因废弃了 但是自己已经写好了  毕竟是自己的第一个node项目 就上传到github 方便自己以后察看;


## Getting Started

```bash
$ git clone https://github.com/liuye1296/eggnode.git
$ cd  eggnode
$ yarn 
$ yarn dev
```
